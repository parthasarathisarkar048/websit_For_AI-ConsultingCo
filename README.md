# XYZ | Data-Driven Insights & AI Solutions

A static marketing website for a boutique AI and analytics firm, highlighting services, industries, team, and a contact flow.

## Features
- Hero section with call-to-action buttons.
- Services grid showcasing four offerings.
- Industries selector with dynamic info card.
- About section with rotating imagery and highlights.
- Leadership team cards.
- Contact form layout.
- Floating chat widget UI.
- Responsive navigation with mobile menu toggle.

## Setup Steps
1. Open `index.html` directly in a browser, or serve the folder with any static file server.
2. If serving locally, ensure the `css/`, `js/`, and `assets/` folders remain alongside `index.html`.
3. Internet access is required for Google Fonts and Lucide Icons (loaded via CDN).

## Project Structure
- `index.html`: Main HTML page and content sections.
- `css/`: Stylesheets.
- `js/`: JavaScript for interactions.
- `assets/`: Images and other static assets.
